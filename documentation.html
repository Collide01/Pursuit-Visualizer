Noteworthy

  • Hamburger icon appears at window sizes of less than 1024px
  • Bulma used for buttons and controls layout
  • Fetch used for preloading presets of controls
  • Audio was turned into a class so multiple different sounds can play at once
  • GIFs animate on the canvas via spritesheets
  • Controls available in full screen by placing everything in a div and setting that to full screen
  • All elements in canvas and controls scale with the size of the canvas so they work at any size
  • Buttons created using shapes and images in the canvas and the mouse position
  • Second canvas used for a menu that displays selectable songs
  • Custom speedlines created by creating a class for them in utils.js
  • Visualizer effects are adjusted with the volume (speed of speedlines, size of bars and circles, etc.)
